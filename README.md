# MurderMystery Utils - Minecraft Forge Mod (1.12.2)

## English Version

### Description
**MurderMystery Utils** is a utility mod that provides a set of helpful features to enhance **Murder Mystery** gameplay in **Minecraft 1.12.2**.

### Features / Modules
| Module | Description |
|--------|-------------|
| **AntiBlind** | Removes blindness effect. |
| **AutoNext** | Automatically joins the next game (/next) for certain roles; can auto-join at the end of a game. |
| **AutoRoleAnnounce** | Automatically sends your role to /party chat (/pc). |
| **BarrierVision** | Barrier visibility (partial implementation). |
| **BowESP** | Displays bows. |
| **CustomTime** | Customize world time. |
| **CustomWeather** | Change weather. |
| **ESP** | Shows players. |
| **FakeGM1** | Fake creative mode. |
| **FlowerPotESP** | Shows flower pots. |
| **Fly** | Enables flight mode. |
| **HUD** | Displays killer/detective info. |
| **ItemESP** | Shows items. |
| **NameTags** | Displays player skin/name/ping. |
| **ShowNames** | Restores hidden player names. |
| **Spammer** | Spam messages from `spam.txt` with support for {player}, {player2}, etc. |
| **Sprint** | Automatic sprinting. |
| **Tracers** | Draws lines to players. |

### Installation
1. Make sure you have **Minecraft 1.12.2** with **Forge** installed.  
2. Download the latest version of **MurderMystery Utils**.  
3. Move the mod `.jar` file to your Minecraft `mods` folder.  
4. Launch Minecraft using **Forge**.

## Русская версия

### Описание
Мод **MurderMystery Utils** предоставляет набор полезных функций для улучшения игрового процесса в **Murder Mystery** на **Minecraft 1.12.2**.

### Функции / Модули
| Модуль | Описание |
|--------|---------|
| **AntiBlind** | Убирает эффект ослепления. |
| **AutoNext** | Автоматический переход в следующую игру (/next) при определённых ролях; можно включить автоматический переход в конце игры. |
| **AutoRoleAnnounce** | Автоматическая отправка вашей роли в /party chat (/pc). |
| **BarrierVision** | Видимость барьеров (частичная реализация). |
| **BowESP** | Отображение лука. |
| **CustomTime** | Настройка времени в мире. |
| **CustomWeather** | Настройка погоды. |
| **ESP** | Отображение игроков. |
| **FakeGM1** | Фейк креатив. |
| **FlowerPotESP** | Отображение цветочных горшков. |
| **Fly** | Режим полёта. |
| **HUD** | Отображение информации убийцы/детектива. |
| **ItemESP** | Отображение предметов. |
| **NameTags** | Отображение скин/ник/пинг игроков. |
| **ShowNames** | Восстановление скрытых никнеймов. |
| **Spammer** | Спам-сообщения из файла `spam.txt` с поддержкой {player}, {player2}… |
| **Sprint** | Автоматическое ускорение. |
| **Tracers** | Линии к игрокам. |

### Установка
1. Убедитесь, что у вас установлен **Minecraft 1.12.2** с **Forge**.  
2. Скачайте последнюю версию мода **MurderMystery Utils**.  
3. Переместите файл `.jar` мода в папку `mods` вашего Minecraft-установщика.  
4. Запустите Minecraft с использованием **Forge**.
